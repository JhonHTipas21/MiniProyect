# 📘 Guía del Usuario - Gestor de Tareas

Bienvenido al Gestor de Tareas 📝. Esta guía te enseñará de forma rápida y sencilla cómo utilizar la aplicación para organizar tus actividades.

---

##  ¿Qué puedes hacer con este programa?

- Agregar nuevas tareas.
- Consultar las tareas existentes.
- Editar tareas ya creadas.
- Eliminar tareas que ya no necesites.
- Organizar tus pendientes para ser más productivo.

---

##  ¿Cómo abrir la aplicación?

1. **Haz doble clic** sobre el archivo `main.py` si tienes Python instalado.
2. O bien, **abre una terminal** y escribe:

   ```bash
   python main.py
   ```

   Asegúrate de estar dentro de la carpeta `MiniProyect-main` para poder ejecutar el main.py.

---

##  Navegación en la interfaz gráfica

Una vez abierta la aplicación, verás una interfaz sencilla con los siguientes elementos:

- **Campos de texto**: Aquí escribes el título de la tarea, descripción y fecha.
- **Botón “Agregar tarea”**: Añade la tarea a la lista.
- **Lista de tareas**: Muestra todas tus tareas actuales.
- **Botón “Eliminar”**: Selecciona una tarea de la lista y presiona este botón para borrarla.
- **Botón “Editar”**: Modifica una tarea existente.

> **⚠️¡OJO!⚠️** 

    ✅ Si dejas el campo de texto vacío e intentas agregar una tarea, el sistema no dejará realizar ninguna accion hasta que completes todos los campos de informacion requerida.

---

##  Consejos útiles

- **Guarda cambios frecuentemente** si haces muchas modificaciones.
- Usa nombres claros para tus tareas (ej. "Estudiar para examen de Sistemas Operativos").
- Elimina tareas completadas para mantener todo limpio y organizado.

---

## 🆘 ¿Problemas?

- Si no puedes abrir el programa, asegúrate de tener instalado **Python** y la librería **PyQt5**.
- Consulta la guía técnica `INSTALACION_Y_EJECUCION.md` para solucionar errores comunes.

---

¡Ahora sí! Con esta informacion, ya estas listo para gestionar de manera organizada tus tareas diarias o periodicas
